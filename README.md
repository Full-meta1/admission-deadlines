#Course Description

## Displays WSOM Admission Deadlines

1. Open terminal in 'admission-deadlines' directory and run 'npm i' to install required node_modules.
2. In terminal, use 'npm start' to initialize development server
3. To use app, the first URL segment is for the school, and the second is for the Program name(i.e. http://localhost:3000/weatherhead/accountancy).



When using with Drupal, App.js must be omitted. <AdmissionDeadlines /> component should be imported and attached directly to index.js. Run "npm run build" to create production build, then proceed to follow "react paragraph" markdown instructions.
